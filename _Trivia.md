
# Ctrl key combo notation
Press and hold `CTRL` then press `X`
- `^X` caret notation
- `C-x` Emacs/Vim notation
- Microsoft
	- `CTRL-X` old Microsoft notation
	- `Ctrl+X` Current Microsoft Notation
- `Ctrl/X` OpenVMS notation
